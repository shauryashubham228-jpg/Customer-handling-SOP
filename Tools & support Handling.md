This document outlines the protocols for handling various customer queries, escalation paths, and standard reply to templates.

## 🛠 Detailed Procedures

### 📦 Order Status

**Questions to Ask:** Order ID, Call from agent? Express/Standard? Address.

1. Check **Shopify** status.
2. If unfulfilled → check with warehouse team.
3. If fulfilled → track order via **DTDC/Delhivery**.
4. Identify stuck/failed scans.
5. Update customer.

> **Reply Template:** *Hey ma'am, I’ve checked your order and it shows [Status]. I’ve coordinated with the courier team and will update you shortly.*
> 

### ⏳ Preorder / Restock

1. Confirm restock approval from vendor.
2. Add stock in Shopify.
3. Inform customer quickly.
4. **Important:** Disable COD (Preorders are non-COD).

> **Reply Template:** *Ma'am, the preorder will be restocked soon. Once available, you can place the order. Preorders are non-COD and refundable only as store credits.*
> 

### 🔄 RTO (Return to Origin)

**Questions to Ask:** Order ID, any call received?

1. Verify RTO status in Shopify.
2. Check arrival in EasyEcom.
3. Conduct QC check after arrival.
4. Process refund (COD → Store Credit only).

> **Reply Template:** *Your order was returned due to delivery issues. Refund will be processed after QC once it reaches our warehouse.*
> 

### 🚚 Return Pickup Issues

1. Check pickup assignment in **Returns Prime**.
2. Verify if an attempt was made.
3. If no attempt → escalate to courier.
4. Request a new attempt.

> **Reply Template:** *We’ve requested a pickup reattempt ma'am. Kindly keep the parcel ready.*
> 

### 📏 Size & Damage Issues

- **Size:** Ask customer to initiate return via [Return Link](https://lagorii.com/apps/return_prime).
- **Damage:** Collect photo/video proof immediately. Mark as "Damaged" in Returns Prime.

> **Reply Template (Damage):** *We’re sorry for this inconvenience ma'am. Please raise the return request; refund will be processed after QC.*
> 

### 💰 Refunds

1. **Check Type:** Prepaid (Original Source) vs. COD (Store Credit).
2. Verify warehouse arrival via EasyEcom.
3. Perform QC.
4. Initiate refund.

> **Reply Template:** *Your refund will be processed after QC once the product reaches our warehouse. COD refunds are issued as store credits.*
> 

### 💳 Store Credit / Gift Cards

**Guide steps for customer:**

1. Go to checkout.
2. Click 'Have a gift card?'.
3. Enter code.
4. System auto-applies credit.

> **Reply Template:** *Please enter your gift card at checkout. It will apply automatically.*
>
